===== משחק "אמת או אמת" - הוראות שימוש =====

שלום שליו!

הנה הקבצים שהכנתי עבורך להעלאת המשחק לאינטרנט:

1. index.html - קובץ המשחק עצמו (הגרסה האחרונה והמעודכנת ביותר)
2. README.md - מידע כללי על המשחק (יוצג באתר GitHub)
3. GITHUB_HOSTING.md - הוראות מפורטות להעלאת המשחק לשרת GitHub Pages
4. play-local.bat - קובץ להפעלה מהירה של המשחק במחשב שלך (פשוט לחץ עליו פעמיים)

== איך להעלות את המשחק לאינטרנט? ==

1. קרא את הקובץ GITHUB_HOSTING.md שמכיל הוראות מפורטות צעד אחר צעד
2. עקוב אחר ההוראות ליצירת חשבון ב-GitHub והעלאת הקבצים
3. לאחר העלאת הקבצים, תקבל כתובת אינטרנט שתוכל לשתף עם חברים

== איך לשחק עם חברים מרחוק? ==

1. שתף את כתובת האתר עם חבריך
2. פתחו שיחת וידאו או אודיו (זום, וואטסאפ, וכו')
3. אחד המשתתפים יפעיל את המשחק ויחלוק את המסך שלו
4. שאר המשתתפים יצפו במסך המשותף וישתתפו בניחושים

== איך לבדוק את המשחק לפני העלאה? ==

פשוט לחץ פעמיים על הקובץ play-local.bat והמשחק ייפתח בדפדפן שלך.

בהצלחה ותהנו מהמשחק!
