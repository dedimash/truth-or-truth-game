# הוראות להעלאת המשחק לשרת GitHub Pages

להלן הוראות מפורטות כיצד להעלות את המשחק לשרת GitHub Pages כדי שחברים יוכלו לשחק בו מרחוק:

## שלב 1: יצירת חשבון GitHub

1. היכנסו לאתר [GitHub](https://github.com/) ולחצו על Sign Up
2. מלאו את הפרטים הנדרשים ליצירת חשבון חדש
3. אשרו את כתובת האימייל שלכם

## שלב 2: יצירת מאגר (Repository) חדש

1. לאחר התחברות לחשבון, לחצו על כפתור "+" בפינה העליונה ובחרו "New repository"
2. תנו למאגר שם, למשל: `truth-or-truth-game`
3. השאירו את המאגר כ-Public (ציבורי)
4. לחצו על "Create repository"

## שלב 3: העלאת קבצי המשחק למאגר

### אפשרות 1: שימוש בממשק האתר (הכי פשוט)

1. בדף המאגר החדש, לחצו על "uploading an existing file"
2. גררו את כל הקבצים מתיקיית `truth-or-truth-game` (index.html ו-README.md) לאזור ההעלאה
3. לחצו על "Commit changes"

### אפשרות 2: שימוש ב-Git (למתקדמים)

אם אתם מכירים את Git, תוכלו להשתמש בפקודות הבאות:

```
git clone https://github.com/YOUR_USERNAME/truth-or-truth-game.git
cd truth-or-truth-game
# העתיקו את הקבצים לתיקייה זו
git add .
git commit -m "Initial commit"
git push origin main
```

## שלב 4: הפעלת GitHub Pages

1. בדף המאגר, לחצו על "Settings" (בתפריט העליון)
2. גללו למטה עד לסעיף "GitHub Pages"
3. תחת "Source", בחרו בענף "main" ולחצו על "Save"
4. המתינו כדקה והרעננו את הדף
5. תופיע הודעה: "Your site is published at https://YOUR_USERNAME.github.io/truth-or-truth-game/"

## שלב 5: שיתוף המשחק עם חברים

1. העתיקו את הקישור שהתקבל בשלב הקודם (https://YOUR_USERNAME.github.io/truth-or-truth-game/)
2. שלחו את הקישור לחברים דרך וואטסאפ, מסנג'ר או כל אמצעי אחר
3. כל חבר יכול לפתוח את המשחק בדפדפן שלו

## איך לשחק יחד מרחוק?

1. פתחו שיחת וידאו או אודיו בזום, וואטסאפ, או כל פלטפורמה אחרת
2. כל המשתתפים פותחים את המשחק בדפדפן שלהם
3. בחרו שחקן אחד שיפעיל את המשחק ויחלוק את המסך שלו
4. שאר השחקנים צופים במסך המשותף ומשתתפים בניחושים
5. אפשר גם לשחק לפי תור, כאשר בכל סיבוב שחקן אחר מפעיל את המשחק

## עדכון המשחק בעתיד

אם תרצו לעדכן את המשחק בעתיד:
1. ערכו את הקבצים במחשב שלכם
2. העלו את הקבצים המעודכנים למאגר ב-GitHub (באותה דרך שהעליתם אותם בהתחלה)
3. האתר יתעדכן אוטומטית תוך מספר דקות
